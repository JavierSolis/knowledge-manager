Skill Manager

CLI interactivo para instalar AI agent skills desde un repositorio local hacia tu proyecto

npx skill-manager init
GitHub Documentación

¿Por qué Skill Manager?

🚀 Repositorio Local

Tus skills, tu control. El repositorio es una carpeta local en tu máquina, no un registry centralizado.

⚡ Búsqueda FTS5

Indexación full-text con SQLite FTS5. Búsqueda rápida por nombre, categoría, tags y descripción.

🔒 Seguridad

Verificación SHA-256 de cada skill instalado. Lock file para tracking de versiones y hashes.

🤖 Multi AI Client

Soporte para Claude, Gemini CLI y más desde el arranque. La ruta de instalación se adapta al AI client.

📦 npm

Un solo comando: npx skill-manager. Sin instalación global, sin configuraciones complejas.

🔄 Actualizaciones Seguras

Detección de cambios por hash/version, backup automático, y nunca chanca sin confirmación.

¿Cómo funciona?

1

Configurá tu repositorio

Apuntá a tu carpeta de skills con skill-manager init

2

Buscá skills

Usá FTS5 para encontrar lo que necesitás

3

Instalá con un click

Seleccioná y confirmá — los archivos se copian automáticamente

Empezá ahora

npx skill-manager init

Sin registro, sin cloud, sin vueltas.
