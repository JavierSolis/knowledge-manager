# Comandos CLI

## skill-manager init

Configuración inicial del repositorio de skills y AI client.

```bash
skill-manager init
```

## skill-manager search

Búsqueda interactiva de skills con FTS5.

```bash
skill-manager search
```

## skill-manager install

Instalar skills seleccionados.

```bash
skill-manager install
skill-manager install --yes       # Sin confirmaciones
skill-manager install --dry-run   # Solo simular
```

## skill-manager list

Listar skills instalados.

```bash
skill-manager list
```

## skill-manager update

Actualizar skills instalados a nuevas versiones.

```bash
skill-manager update
```

## skill-manager rescan

Re-escanear el repositorio de skills para refrescar el caché.

```bash
skill-manager rescan
```
